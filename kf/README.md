## File commands


### Usage:

- Example:
```
kf somefile.txt --> creates a new file named somefile.txt
kf create somefile.txt --> creates a new file named somefile.txt (alias: c)
kf somefile.txt somefile2.txt --> renames somefile.txt to somefile2.txt
kf rename somefile.txt somefile2.txt --> renames somefile.txt to somefile2.txt (alias: r)
```
